Installation
============

Instructions
------------

To install, install via pip or easy_install::

    $ pip install newrelic-api
    or
    $ easy_install newrelic-api

If you want to install it from source, run::

 $ pip install git+git://github.com/ambitioninc/newrelic-api.git


Requirements
------------

* requests>=2.0.0
* Python 2.7, 3.3, 3.4
